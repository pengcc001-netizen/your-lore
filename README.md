# Your Lore — Personal Story & Character Backstory Generator

Personal story generator that creates your character lore and backstory. Generate RPG-style backstories from your traits.

**Live site: [https://lore.csskey.com](https://lore.csskey.com)**

## Features

- **Character backstory generator** — Build lore from your answers
- **RPG-style output** — Class, origin, and quest
- **Trait-based lore** — Personality drives the story
- **Save and share** — Export your character lore
- **Multiple genres** — Fantasy, sci-fi, and modern

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS
- **SEO:** react-helmet-async, JSON-LD Schema
- **Hosting:** Cloudflare Workers

## Quick Start

```bash
# Install dependencies
bun install

# Start dev server
bun run dev

# Build for production
bun run build
```
